# mysql2h2-plus
借鉴已开源的技术，不断完善转换的IDEA 插件

可以打包后直接拖入IDEA并使用 / 或直接下载 https://github.com/ChenYilei2016/mysql2h2-plus/blob/master/build/distributions/mysql2h2-plus-1.0.0.zip

功能:  
1 mysql 转换 h2表结构  
2 兼容索引外部create Index方式  
3 兼容索引表中携带方式  
3 兼容没有分号的情况  
4 兼容h2索引名唯一性  
5 去除create表后的属性  
6 兼容json
